# NAMU - Creative Portfolio Concept

NAMU is a high-end, aesthetic portfolio website designed for creative professionals. This project showcases a modern approach to web design, focusing on clean typography, a sophisticated color palette, and a seamless user experience.

## ✨ Project Highlights

* **Premium UI/UX:** Features a modern glassmorphism contact form and elegant section badges.
* **Fully Responsive:** Optimized for all screen sizes, from high-resolution desktops to mobile devices.
* **Performance Focused:** Built with semantic HTML5 and optimized CSS3 for fast loading and SEO friendliness.
* **Modular Code:** Uses CSS variables for easy branding updates and theme management.

## 🛠️ Built With

* **HTML5:** Semantic structure for better accessibility and SEO.
* **CSS3:** Advanced layouts using Flexbox and CSS Grid.
* **Google Fonts:** Utilizing "Poppins" and "Inter" for a clean look.
* **GitHub Pages:** Hosting and deployment.

## 📁 Project Structure

- `index.html` - The main architectural structure of the site.
- `css/style.css` - Global styles, variables, and main layouts.
- `css/components.css` - Reusable UI elements (Buttons, Cards, Inputs).
- `images/` - Optimized assets and project previews
